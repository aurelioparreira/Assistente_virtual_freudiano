# Assistente_virtual_freudiano
Laboratório experimental que une Whisper (Speech-to-Text), ChatGPT (IA conversacional) e gTTS (Text-to-Speech) para criar um assistente de voz multi-idiomas inspirado na psicanálise freudiana. O projeto integra o acervo das Obras Completas de Freud para oferecer respostas fundamentadas em conceitos psicanalíticos.
